# Finance Analyzer ETL Pipeline (Пет-проект)

Анализ личных финансов с использованием продвинутых методов Pandas. Проект эволюционировал в полноценный ETL-конвейер.

## Технические приемы:

**1.** Расчет скользящих средних для сглаживания трат (через rolling).

**2.** Быстрый расчет отклонений от среднего.

**3.** Группировка данных на нескольких уровнях для получения бизнес-метрик.


### Что добавлено:
**1)** Полная контейнеризация связки Python + PostgreSQL через Docker Compose.

**2)** Переход от CSV-хранилища к PostgreSQL с использованием SQLAlchemy.

**3)** Расширенная логика трансформации данных (анализ долей бюджета, скользящие метрики).

**4)** Настроено детальное логирование всех этапов работы конвейера.

## Технологический стек
* **Python 3.13 (Pandas, Numpy, SQLAlchemy, Psycopg2)**
* **PostgreSQL**
* **Docker & Docker Compose**

## Структура проекта
* `finance_analyzer.py` - основной файл.
* `Dockerfile` - конфигурация образа Docker
* `docker-compose.yml` - App + DB 
* `requirements.txt` - текстовый файл с зависимостями
